## ♟️ 체스 초보자들을 위한 오프닝 강의 검색 사이트 체스몽
- 체스를 하다가 현재 상태에 대한 강의를 찾고 싶을 때, 체스몽의 체스판에서 검색하고 싶은 상태로 말을 두면 맞춤형 유튜브 체스 강의 목록을 확인할 수 있어요!
- PC와 Tablet, Mobile 기기 제한 없이 사용할 수 있어요!
